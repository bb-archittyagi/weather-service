= Weather Service (Vert.x)
:toc:
:toclevels: 2

A reactive backend weather service built using *Java* and *Vert.x*, designed to fetch real-time weather data from an external API, process it asynchronously, and store it in a *MySQL* database.  
The application exposes RESTful APIs and follows a clean, layered architecture suitable for scalable microservices.
